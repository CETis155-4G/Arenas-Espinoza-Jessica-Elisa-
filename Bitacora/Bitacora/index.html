HTML

"

El HTML es la quinta revisión importante del lenguaje básico de la World Wide Web HTML5 es una colección de estándares para diseño y desarrollo de páginas web. Esta colección representa la manera en que se presenta la información en el explorador de internet y la manera de interactuar con ella. HTML es confundido por muchos, como un lenguaje de programación, cuando en realidad HTML es un lenguaje de marcado. Los lenguajes de marcado en pocas palabras se utiliza para estructurar documentos y su información mediante el uso de ETIQUETAS. HTML5 es una versión estándar del programa, con estándar quiere decir que es sólo el patrón de referencia, sin enbargo, no es un programa profecional. se maneja a base de etiquetas y sirve principalmente para el diseño de paginas web junto con CSS y JavaScript

CSS

"

La css conecta con el html Sus siglas corresponden a “Cascading Style Sheets”, que tiene el siguiente significado: Cascading, que significa que los estilos que aplicamos a los elementos de una página web se propagan a los elementos que contiene, se propagan en cascada. Es un lenguaje modelado o un lenguaje de diseño gráfico que en español a "hojas de estilo cascada" y se le denomina asi porque la caracteristica se aplica de arriba a abajo. Es un lenguaje que complementa al tipado antes mencionado HTML y principalmente sirve como un idioma que nos permite darle una mejor interfaz a nuetra pagina web desde el uso de colores, tipo de fuente, tamaño, etc.
